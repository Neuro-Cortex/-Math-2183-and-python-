## 🚀 Features

- **Matrix Operations**
  - Addition, subtraction, multiplication
  - Determinant, inverse, rank
  - Visualization of matrix multiplication
- **Vector Operations**
  - Vector addition, scalar multiplication
  - Dot product, cross product
  - Projection, orthogonality visualization
- **Eigenvalues & Eigenvectors**
  - Power method implementation
  - Diagonalization of matrices
  - Visualization of eigenvectors
- **Linear Transformations**
  - 2D/3D transformation visualization





** Features
- Matrix operations (add, multiply, inverse, determinant)

- Eigenvalues & Eigenvectors (power method, diagonalization)
